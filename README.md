# Approval , Notification  Addons

## Overview

This repository contains custom Odoo addons for Approval Management Notification and Delegation of Authority (DOA).

Modules included:

1. `approval_template`
2. `approval_instance`
3. `approval_task_line`
4. `approval_task`
5. `approval_doa`
6. `approval_matrix`
7. `notification`
